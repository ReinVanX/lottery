## 一、两张海报的共同视觉 DNA

两张海报属于同一套 **2026 医焕安全球药物警戒年会 · 中国站** 视觉体系，差异主要在文案侧重，风格高度一致。

### 1. 整体气质


| 维度    | 海报特征                              |
| ----- | --------------------------------- |
| 风格关键词 | 高端会务、科技医疗、全球连接、庄重权威               |
| 情绪    | 深色沉稳 + 金色高光，偏「正式晚宴/国际峰会」，而非「霓虹赛博」 |
| 构图    | 左文右信息、中间城市天际线锚点、右上/左下光效平衡         |
| 层次    | 摄影实景（上海）+ 线稿插画（北京长城）+ 3D 图形（地球）混合 |


### 2. 色彩体系（核心差异点）

海报主色不是青绿，而是 **深海军蓝 + 琥珀金**：


| 角色   | 海报近似色                             | 用途              |
| ---- | --------------------------------- | --------------- |
| 背景深底 | `#050B18` ~ `#0A1628`             | 大面积深色，近黑海军蓝     |
| 背景渐变 | 深蓝径向光晕                            | 左上地球区、底部粒子区有暖光  |
| 主强调色 | `#E8A838` / `#F5B84A` / `#FF9D00` | 标题、日期、分隔线、图标、光轨 |
| 次强调色 | `#FFFFFF`                         | 正文、副标题、Logo 文字  |
| 辅助光色 | 淡橙金光晕 `rgba(255,180,60,0.4)`      | 标题发光、粒子轨迹       |
| 城市灯光 | 蓝白 + 金边高光                         | 天际线建筑灯光         |


**当前抽奖页主色是青绿系（**`#00C2CB` **/** `#00E5EE`**）**，与海报金色体系偏差最大，这是首要改造点。

### 3. 字体与排版


| 层级        | 海报做法                           | 当前页面                   |
| --------- | ------------------------------ | ---------------------- |
| 主标题       | 大号 **金色**，部分用 **衬线体**（如「演讲嘉宾」） | 白色 + 青色高亮，全 sans-serif |
| 品牌 Slogan | 「智联全球 警戒未来」金色、字距宽、权威感          | 灰色/青色，偏小               |
| 日期/地点     | 金色粗体，右侧竖向信息栏                   | 无对应结构                  |
| 正文        | 白色，14–16px，克制                  | 蓝灰色 `--text-dim`       |
| Logo      | NEXTROVE 白色无衬线 + 图形标           | 有，但配色不统一               |


**建议：** 主标题/奖项名用金色；正文白色；Slogan 用金色或浅金；衬线体仅用于 1–2 处大标题（不宜全文使用）。

### 4. 核心图形元素

```
┌──────────────────────────────────────────────────────┐
│  [地球+轨道线]                    │  日期/城市信息栏  │
│                                   │  (金色竖线分隔)   │
│  主标题（金色）                    │  2026.09.08      │
│  活动全称                          │  中国·上海        │
│  Slogan                            │  ─────────       │
│  描述文案                          │  2026.09.10      │
│  [四个金色线稿图标]                │  中国·北京        │
│                                   │                  │
│         [上海天际线 + 水面倒影]     │  [长城线稿]       │
│  ~~~~~~~~金色粒子光轨~~~~~~~~      │                  │
│  NEXTROVE | 活动全称               │                  │
└──────────────────────────────────────────────────────┘
```

需要在抽奖页背景中体现的三类元素：

1. **地球 + 轨道网络**（左上）— 表达「全球药物警戒」
2. **城市天际线**（中下）— 上海实景感，年会地点氛围
3. **金色粒子光轨**（底部）— 动态能量感

当前页面只有：青色电路 SVG、青色水印、青绿粒子，**缺少地球、天际线、金色光轨**。

## 二、当前抽奖页面风格审计



### 现有设计语言（NEXTROVE 第一版改造后）

```css
/* 当前核心令牌 */
--teal: #00C2CB;
--teal-bright: #00E5EE;
--gold: #FFD966;          /* 仅用于中奖，非主色 */
background: linear-gradient(180deg, #0A4D6E → #001428);
```


| 模块    | 现状                 | 与海报匹配度         |
| ----- | ------------------ | -------------- |
| 背景渐变  | 青蓝竖向渐变             | 低 — 海报偏深黑金     |
| 强调色   | 青绿 everywhere      | 低 — 应改为金色      |
| 毛玻璃面板 | 青色边框+模糊            | 中 — 形态可保留，色温需改 |
| 角括号框  | 青色发光               | 中 — 可改金色细线框    |
| 水印    | 青色 NEXTROVE / 2026 | 低 — 应改淡金色      |
| 电路装饰  | 青色 SVG 线           | 低 — 应改金色粒子轨    |
| 粒子动画  | 青蓝粒子               | 低 — 应改金/暖色     |
| 抽奖滚动条 | 青色描边               | 低 — 揭晓时用金色合理   |
| 品牌区   | 白色+青色 slogan       | 中 — 缺海报式信息栏    |


**结论：** 结构框架（毛玻璃、对角水印、四页流程）可保留；**色温体系需从「青绿科技」整体切换为「深金会务」**。

## 三、差距总结（Gap Analysis）



### 必须改（P0）

1. **主色板**：青绿 → 琥珀金
2. **背景**：改为深海军蓝 + 局部暖光晕（非青蓝渐变）
3. **标题/按钮/边框发光**：统一金色体系
4. **文案色**：副文从 `#8BB8CC` 改为 `#C8D0E0` 或白色 80% 透明度



### 应该改（P1）

1. **背景加入地球 + 天际线**（静态或轻动态）
2. **底部金色粒子光轨**（替代青色电路线）
3. **右侧信息栏结构**（日期/城市，可选）
4. **Slogan 视觉权重提升**（金色、更大字距）



### 可以改（P2）

1. 天际线灯光呼吸动画
2. 地球轨道线缓慢旋转
3. 水面倒影微动
4. 衬线体主标题

## 四、推荐新设计令牌（Design Tokens）

建议在 `:root` 中替换/新增：

```css
:root {
  /* 背景 */
  --bg-deep:    #050B18;
  --bg-mid:     #0A1628;
  --bg-warm:    rgba(255, 160, 50, 0.06);   /* 暖光晕 */

  /* 主强调 — 海报金 */
  --gold:       #E8A838;
  --gold-bright:#F5C842;
  --gold-deep:  #C88A20;
  --gold-glow:  rgba(255, 180, 60, 0.45);

  /* 文字 */
  --text:       #FFFFFF;
  --text-dim:   rgba(255, 255, 255, 0.72);
  --text-muted: rgba(255, 255, 255, 0.45);

  /* 玻璃面板 — 暖色边框 */
  --glass-bg:     rgba(5, 11, 24, 0.35);
  --glass-border: rgba(232, 168, 56, 0.22);
  --glass-shadow: inset 0 0 48px rgba(255, 180, 60, 0.04),
                  0 8px 40px rgba(0, 0, 0, 0.35);

  /* 保留青绿仅用于极少辅助（如链接），或完全移除 */
}
```



### 背景渐变建议

```css

  background:

    linear-gradient(180deg, #0A1628 0%, #050B18 55%, #030810 100%);
}
```

## 六、背景元素实现方案（地球 + 天际线 + 动态感）



### 方案对比


| 方案              | 优点       | 缺点        | 推荐        |
| --------------- | -------- | --------- | --------- |
| A. 静态切图         | 最快、最接近海报 | 无动态、文件较大  | 第一阶段      |
| B. CSS 分层 + 轻动画 | 单文件、可控   | 地球/天际线需素材 | **推荐主方案** |
| C. Canvas 绘制    | 动态最强     | 开发量大      | 粒子和光轨部分   |




### 推荐布局（`#bgDecor` 内）

```
z-index 0 层（从后到前）：
  1. body 渐变底色
  2. .bg-globe        — 左上，宽 28vw，opacity 0.55
  3. .bg-orbit-lines  — 地球周围金色轨道 SVG，CSS rotate 动画
  4. .bg-skyline      — 底部居中，宽 70vw，贴底
  5. .bg-skyline-glow — 天际线顶部金色光晕遮罩
  6. .bg-particles    — 底部金色粒子轨（Canvas 或 CSS）
  7. .watermark-brand / .watermark-year
  8. 毛玻璃面板 / 抽奖 UI（z-index 1+）
```



### 素材来源建议

1. **天际线**：从海报提取上海天际线（需去文字），或使用免版权夜景素材，叠加 `mix-blend-mode: screen` 或金色 `filter` 统一色温
2. **地球**：海报左上地球裁切为 PNG（透明底），或 SVG 线稿地球
3. **长城**（可选）：右下角淡金色线稿，opacity 0.15，不抢主体



### 动态效果设计（呼应海报）


| 效果      | 做法                                           | 强度       | 性能  |
| ------- | -------------------------------------------- | -------- | --- |
| 地球轨道旋转  | SVG `transform: rotate` 120s linear infinite | 极慢       | 低   |
| 天际线灯光呼吸 | CSS `@keyframes` 改变 `opacity` + `brightness` | 3–4s 周期  | 低   |
| 水面倒影波动  | `transform: scaleY` 微幅 + `opacity` 脉动        | 极 subtle | 低   |
| 底部金粒子轨  | 复用 `ParticleEngine`，改金色，沿贝塞尔曲线漂移             | 中等       | 中   |
| 地球节点闪烁  | Canvas 小亮点 random alpha                      | 可选       | 中   |


**注意：** 抽奖滚动时 Canvas 已有动画，背景动态应保持 **慢、淡、不抢眼**，避免干扰名单阅读。

| 组件     | 现状                        | 建议                          |

| 爆发粒子   | `#FFD966`                 | 保持金色，提高亮度                   |
| 滚动卡片边框 | 青色                        | 白色/淡金；中奖金色                  |
| 中线指示   | 青色                        | 金色                          |
| 遮罩层    | `rgba(0,20,40,0.55)`      | `rgba(5,11,24,0.5)`         |

## 九、风险与约束


| 风险           | 说明               | 应对                     |
| ------------ | ---------------- | ---------------------- |
| 素材版权         | 海报摄影/合成图需确认商用范围  | 优先裁切自有海报素材             |
| 文件体积         | 天际线 PNG 可能较大     | WebP + 适当压缩，目标 < 500KB |
| 大屏可读性        | 金色+深色对比需足够       | 正文保持白色，金色仅强调           |
| 性能           | 多层动画 + Canvas 抽奖 | 背景动画降帧，抽奖时减弱背景动效       |
| `file://` 离线 | 单 HTML 需内联或相对路径  | 素材放 `assets/`，相对引用     |

## 十、总结与优先级建议

**一句话：** 当前页面是「青绿科技风」，海报是「深海军蓝 + 琥珀金会务风」——结构可留，色温与背景元素需重做。

**若只做一件事：** 先把全站主色从青绿改为海报金色体系（Phase 1），视觉匹配度可从约 30% 提升到约 70%。

**若要做到高度还原：** Phase 1 + 2 + 3，加入地球、天际线及轻动态，可达约 90% 品牌一致性。
