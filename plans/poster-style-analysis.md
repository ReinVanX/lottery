## 1. 色彩体系分析

### 1.1 海报配色（目标风格）

通过 K-Means 聚类和直方图分析，提取海报的核心色板：

| 色号 | 占比 | 用途推断 |
|------|------|----------|
| `#000019` | **62-73%** | 🏴 主背景 — 极深海军蓝/近乎黑色 |
| `#000032` | 4.5-6.2% | 深蓝过渡层 |
| `#191919` | 5.1-6.1% | 深灰色调 — 建筑/剪影暗部 |
| `#001932` | 3.0-5.8% | 暗蓝渐变过渡 |
| `#4B3219` | 1.3-1.8% | 🟤 **暖色点缀** — 深青铜/古铜色（天际线高光） |
| `#321919` | 1.5-2.6% | 暗红棕 — 暖色辅助 |
| `#32324B` | 1.3-2.3% | 蓝灰色中间调 |
| `#646464` | 0.6-0.7% | 灰色高亮点 |

**色调特征**:
- 🔵 **主导通道**: 蓝色（R=15-19, G=26-29, B=46-50）
- 🌡️ **冷暖倾向**: 略微偏冷（Warmth Index = -31），但有暖色平衡
- 🎯 **饱和度**: **82-86%**（极高饱和度，色彩浓郁）
- 🌑 **整体亮度**: **25-29**（极暗，接近纯黑）

### 1.2 当前设计配色

| CSS 变量 | 色值 | 用途 |
|----------|------|------|
| `--bg-deep` | `#001428` | 深蓝背景 |
| `--bg-mid` | `#0A3D5C` | 中蓝背景 |
| `--teal` | `#00C2CB` | 主强调色 |
| `--teal-bright` | `#00E5EE` | 亮青强调 |
| `--blue` | `#0088CC` | 辅助蓝 |
| `--gold` | `#FFD966` | 金色高亮 |
| `--text-dim` | `#8BB8CC` | 次要文字 |
| `--text-muted` | `#5A8FA8` | 更淡文字 |

**色调特征**:
- 🔵 **主导通道**: 蓝色（R=61, G=111, B=147）
- 🌡️ **冷暖倾向**: 明显偏冷（Warmth Index = -86）
- 🎯 **饱和度**: **61%**（中等饱和度）
- 🌓 **整体亮度**: **100**（中等偏暗）

### 1.3 🔴 核心差距

| 维度 | 海报（目标） | 当前设计 | 差距 |
|------|-------------|----------|------|
| 背景基调 | 极暗海军蓝 `#000019` | 暗蓝 `#001428` | 需更深 60% |
| 氛围 | 接近纯黑的神秘感 | 可见蓝色的科技感 | 风格定位不同 |
| 饱和度 | 82-86% 高饱和 | 61% 中等 | 需提升 35% |
| 暖色运用 | 有古铜/深棕点缀 | 仅有金色 | 需增加暖色层次 |
| 强调色 | 隐约的暗金色光晕 | 明亮的青色发光 | 需降低强调色亮度 |

## 2. 亮度与氛围对比

### 2.1 海报亮度分布

```
海报纵向亮度剖面（从上到下）：
y=  0-10%:  Bright=20  ████████████                    ← 顶部极暗
y= 10-20%:  Bright=32  ████████████████████             ← 微亮（可能是标题/Logo区域）
y= 20-30%:  Bright=35  █████████████████████            ← 最亮区域（环球地图/视觉焦点）
y= 30-50%:  Bright=28  █████████████████                ← 中段过渡
y= 50-80%:  Bright=29  █████████████████                ← 均匀暗部
y= 80-100%: Bright=24  ██████████████                   ← 底部最暗（城市天际线剪影）
```

**关键发现**:
- 海报整体亮度在 **20-35** 范围内，极其克制
- 唯一的"亮区"在 20-30% 位置，仅为 Bright=35（仍是非常暗）
- 底部有天际线剪影（检测到亮度跃迁点，距底部约 14-70 像素）

### 2.2 当前设计亮度分布

```
当前设计（poster-reference.png）纵向亮度：
y=  0-20%:  Bright=99-145   ████████████████████████    ← 顶部很亮（天空发光区）
y= 20-40%:  Bright=139-154  ██████████████████████████   ← 中上高亮（内容区）
y= 40-70%:  Bright=62-94    ████████████                 ← 中段迅速变暗
y= 70-100%: Bright=64-86    █████████████                ← 底部暗蓝
```

### 2.3 🔴 关键差距

**当前设计是"顶部亮 → 底部暗"的强对比渐变**，而海报是 **"整体极暗 + 微妙层次"** 的沉稳风格。当前背景渐变：

```css
/* 当前 */
background: linear-gradient(180deg, #0A4D6E 0%, #001428 42%, #000D1A 100%);
```

顶部 `#0A4D6E` 亮度约 90，而海报顶部亮度仅 20。**差距 4.5 倍**。

## 3. 排版与字体风格

### 3.1 海报字体特征（推断）

基于色彩分布和结构分析，海报的字体风格：
- **标题/品牌名**: 大概率使用无衬线粗体（Sans-serif Bold），颜色为亮白或金属色
- **活动名称**: 中文字体，字重较重，可能带 letter-spacing
- **副标题/口号**: 较小字号，颜色偏暗金或灰蓝
- **正文信息**: 极简克制，深灰蓝/暗金小字

### 3.2 当前设计字体

| 元素 | 当前样式 | 问题 |
|------|---------|------|
| 品牌水印 "NEXTROVE" | `rgba(0,194,203,0.045)` 青色调 | 应改为更暗的海军蓝调 |
| 年份水印 "2026" | `rgba(0,194,203,0.04)` | 同上 |
| 标题强调色 | `#00E5EE` 带青发光 | 太亮，应改为暗金/暖白 |
| 按钮主色 | `#00C2CB → #0088CC` 青蓝渐变 | 应改为深青铜/暗金调 |
| 中奖名字 | `#FFD966` 金色 | ✅ 保留，但需降低发光强度 |

## 4. 背景与装饰元素

### 4.1 海报的视觉层次

分析表明海报有以下层次结构：

```
┌─────────────────────────────────────────┐
│  Layer 1: 品牌/活动标识文字（顶部）         │  ← 亮白/金属色
│  Layer 2: 环球地图或地球图形（中上部）       │  ← 暗蓝基调上的微亮区域
│  Layer 3: 会议信息/文字内容（中部）          │  ← 信息密集区
│  Layer 4: 城市天际线剪影（底部）             │  ← 暗色剪影 + 暖色高光线
│  Layer 5: 深海军蓝背景基底                  │  ← #000019 极暗基底
└─────────────────────────────────────────┘
```

### 4.2 当前设计层次

```
┌─────────────────────────────────────────┐
│  Layer 1: 电路纹理 SVG + 圆形装饰          │  ← 青色线条
│  Layer 2: 毛玻璃面板                       │  ← teal 边框 + 玻璃效果
│  Layer 3: Canvas 粒子动画                  │  ← 青/蓝色粒子
│  Layer 4: 品牌水印文字                      │  ← teal 超低透明度
│  Layer 5: 深蓝→黑渐变背景                  │  ← 亮度对比较大
└─────────────────────────────────────────┘
```

### 4.3 🔴 修改方向

| 当前元素 | 问题 | 修改方向 |
|---------|------|---------|
| 电路纹理 SVG | 科技感太强，与海报的沉稳风格不一致 | 替换为天际线剪影 + 地球轮廓 |
| 毛玻璃面板 | 亮度偏高，teal 发光过于突出 | 改为暗色半透明 + 暖金边框 |
| 圆形装饰 | 现代科技感 | 保留但改为暗金/铜色暖调 |
| 水印文字 | 颜色太亮（teal系） | 改为极暗海军蓝调 |

## 5. 动画与粒子系统

### 5.1 当前粒子系统

```javascript
// 当前粒子颜色配置
```

### 5.2 🔴 修改方向

将粒子从 **青蓝色系** 改为 **暗金 + 铜色系**，与海报的暖色点缀呼应：

```javascript
// 建议改为
```

粒子爆发（burst）效果保留金色 `'255, 217, 102'` 作为中奖庆祝，这个和海报表意不冲突。

## 6. 城市天际线与环球地图

### 6.1 动态天际线方案

根据海报底部的天际线剪影特征，建议实现以下方案：

#### 方案 A：SVG 天际线剪影（推荐）

在天际线 SVG 中使用多层视差滚动：

```
┌────────────────────────────────────────────┐
│  远山/远景层（最暗，移动最慢）                  │
│  ▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔                 │
│  中层建筑群（中等暗度，移动速度中等）             │
│  ▓▓▓▓  ▓▓▓▓▓  ▓▓▓  ▓▓▓▓▓▓                │
│  前景建筑群（稍亮，有暖色高光线，移动最快）        │
│  ██  ████  ██  █████  ██                  │
│  暖色高光线（沿建筑顶部轮廓的暗金细线）           │
│  ───────────╱╲──╱╲─────────               │
└────────────────────────────────────────────┘
```

**实现要点**:
- 3 层天际线 SVG，分别设置不同的 `transform: translateX()` 动画速度
- 建筑轮廓使用 `#000019` 到 `#0A1020` 的暗蓝渐变
- 建筑顶部添加 `#4B3219`（古铜色）的细高光线
- 整体透明度 0.15-0.30，融入背景

#### 方案 B：Canvas 动态生成天际线

使用 JavaScript 在 Canvas 上程序化生成建筑轮廓：
- 随机生成不同高度的矩形建筑
- 添加窗户灯光点（暗金色随机分布）
- 缓慢的水平漂移动画

#### 方案 C：CSS 纯实现

使用多个 `box-shadow` 或 `clip-path` 创建静态天际线，配合 CSS 动画实现缓慢移动。

### 6.2 环球地图方案

在背景中添加低调的地球/世界地图轮廓：

**实现方式**:
- SVG 世界地图简化轮廓，使用 `rgba(75, 50, 25, 0.04)` 青铜色调
- 放置于中上部（y: 15%-35% 区间）
- 缓慢旋转动画（使用 CSS `transform: rotate()` 或 Canvas）
- 可选：添加经纬线网格增加地理感

## 9. 天际线 + 地球实现建议

### 9.1 天际线 SVG 结构

```html
<div class="skyline-container" aria-hidden="true">
  <!-- Layer 1: 远景山脉/低矮建筑 — parallax speed 0.2 -->

    <path d="M0,200 L0,120 L80,100 L160,115 L240,95 L320,110 L400,90 ... L1920,130 L1920,200 Z"

  </svg>
```

```css
.globe-outline {
```
