# 音效文件目录

请将以下 2 个 MP3 文件放入此文件夹：

| 文件名 | 用途 | 建议 |
|--------|------|------|
| `draw.mp3` | 抽奖滚动中（循环播放） | ~3-8 秒，紧张悬疑感，如鼓点、心跳、轮盘旋转声 |
| `win.mp3` | 中奖揭晓（单次播放） | ~2-5 秒，辉煌胜利感，如号角、烟花、欢呼声 |

## 音效来源建议

- **免费音效网站**：freesound.org、mixkit.co、pixabay.com/sound-effects
- **搜索关键词**：
  - draw: "suspense" / "tension" / "drum roll" / "spinning wheel" / "countdown"
  - win: "fanfare" / "victory" / "achievement" / "ta-da" / "congratulations"

## 注意事项

- 文件必须是 `.mp3` 格式
- 文件名必须与上表一致
- 如果没有放置某个文件，程序会自动回退到内置合成音效
- `draw.mp3` 会自动无缝循环播放
- 文件放入后刷新页面即可生效
