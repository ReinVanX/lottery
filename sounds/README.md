# 音效文件目录

请将以下 5 个 MP3 文件放入此文件夹：

| 文件名 | 用途 | 建议时长 |
|--------|------|---------|
| `tick.mp3` | 滚动滴答声 | ~0.1-0.3 秒，短促清脆 |
| `hum.mp3` | 滚动嗡鸣声（循环播放） | ~2-4 秒，低音引擎感 |
| `reveal.mp3` | 揭晓冲击声 | ~1-3 秒，有爆发力 |
| `celebrate.mp3` | 庆祝声 | ~2-4 秒，辉煌胜利感 |
| `void.mp3` | 作废/失败声 | ~1-3 秒，下坠消散感 |

## 音效来源建议

- **免费音效网站**：freesound.org、mixkit.co、pixabay.com/sound-effects
- **搜索关键词**：
  - tick: "UI click" / "button tap" / "camera shutter"
  - hum: "engine idle" / "low drone" / "tension riser"
  - reveal: "impact" / "boom" / "reveal" / "whoosh hit"
  - celebrate: "fanfare" / "victory" / "achievement" / "ta-da"
  - void: "fail" / "buzzer" / "error" / "down sweep"

## 注意事项

- 文件必须是 `.mp3` 格式
- 文件名必须与上表一致（区分大小写）
- 如果没有放置某个文件，程序会自动回退到内置合成音效
- 支持 MP3 循环播放（hum.mp3 会自动无缝循环）
- 文件放入后刷新页面即可生效，无需修改代码
